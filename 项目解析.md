## 四、数据流分析

```
用户操作 → Vue 组件 → API 层 (src/api/) → Axios (src/utils/request.js)
                                              │
                              ┌────────────────┴────────────────┐
                              ▼                                 ▼
                     Apifox Mock (远程)                  Mock.js (本地)
                  baseURL: m1.apifoxmock.com           拦截 Ajax 请求直接返回
                              │                                 │
                              └────────────────┬────────────────┘
                                               ▼
                                         组件更新视图
```

### request.js 拦截器机制

```
请求拦截器：
  - 自动从 localStorage 读取 token
  - 注入 Authorization: Bearer <token>

响应拦截器：
  - 自动解包 response.data（组件直接用，无需 .data）
  - 统一错误处理：401/403/404/5xx / 超时 / 网络错误
```

### Mock 策略

- **远程 Mock**：当前 baseURL 指向 Apifox 云端 Mock 服务
- **本地 Mock**：`src/mock/` 下使用 Mock.js 拦截 Ajax 请求，在内存中模拟 CRUD
- **切换方式**：修改 `src/utils/request.js` 中 baseURL 的注释，切换真实后端即可

## 七、技术特点

1. **JavaScript ESNext + ES Modules**：项目主语言为 JavaScript，`package.json` 中 `"type": "module"`，所有文件使用 `import/export` 语法；`jsconfig.json` target 设为 `ESNext`
2. **Vue 3 Composition API**：所有组件使用 `<script setup>` 语法，代码简洁，逻辑清晰
3. **Vue 3 响应式系统**：使用 `ref` / `reactive` / `computed` / `watch` / `onMounted` 等组合式 API
4. **Element Plus 中文国际化**：全局配置 `zhCn` locale
5. **全局图标注册**：Element Plus 所有图标在 `main.js` 中一次性注册为全局组件，模板中直接使用无需 import
6. **Vite 路径别名**：`@ → src/`，同时配置 `jsconfig.json` 的 `paths` 和 `vite.config.js` 的 `alias`
7. **响应式布局**：多处使用 `@media` 查询适配移动端/平板（768px / 900px / 1100px / 1200px 断点）
8. **统一 API 层**：所有请求通过 `request.js` 实例发出，便于切换后端地址
9. **Axios 拦截器链**：请求拦截自动注入 Token，响应拦截自动解包 `response.data` + 统一错误处理
10. **前端分页模式**：列表一次性获取全部数据，筛选和分页在前端计算完成（`computed` + `slice`）
11. **双向 Mock 策略**：Apifox 云端 Mock + Mock.js 本地 Mock，修改 baseURL 即可切换真实后端
12. **Sass/SCSS 样式预处理**：支持变量、嵌套等高级 CSS 写法
13. **ESLint Flat Config**：使用 ESLint 9 全新的 flat config 模式（非传统 `.eslintrc`），集成 Prettier
14. **CSS 动画与过渡**：hover 动效、`transform` 过渡、`@keyframes` 关键帧动画（搜索框弹出、加载 spinner）
